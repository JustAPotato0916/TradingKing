## Getting Started

### Prerequisites

- npm or yarn
  ```sh
  npm install npm@latest -g # or npm install --global yarn
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/JustAPotato-dev/TeamMemberAllocation.git
   ```
2. Install NPM packages
   ```sh
   npm install # or yarn install
   ```
3. Run the project
   ```sh
    npm run dev # or yarn dev
   ```
